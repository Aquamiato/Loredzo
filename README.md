## ⚓ About

**Loredzo** is a luxury resort swimwear and coastal beachwear landing page for the discerning woman. Inspired by Mediterranean coastal living, it showcases precision-cut swimsuits, elegant bikinis, structured cover-ups and high-end vacation fashion — all available through [Aquamiato.com](https://aquamiato.com).
